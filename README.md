# KPR College Certificate Registration System

A complete web application for distributing certificates to registered students via email.

## Features

✅ **Student Registration Verification** - Check if student is registered in database  
✅ **Automatic Certificate Generation** - Generate PDF certificates dynamically  
✅ **Email Distribution** - Send certificates via email with download link  
✅ **Download Tracking** - Log and track certificate downloads  
✅ **Responsive Design** - Works on desktop, tablet, and mobile devices  
✅ **Security** - Input validation, database prepared statements, secure file handling  

## System Requirements

- PHP 7.4+
- MySQL 5.7+
- Apache/Nginx web server
- TCPDF library for PDF generation
- PHPMailer for email sending
